Course Timeline

Week 1: Introduction to Python & Development Environment

History, Python 3.13 installation, virtual environments, IDEs, REPL, `pip`, Hello World script, Python REPL, environment setup, and workspace configuration in VS Code.

Week 2: Variables, Data Types & Basic I/O

Variables, naming conventions, primitive types, type hints, string encoding, user input, type casting, `id()`, `type()`, `is`, `==`, and hands-on converters.

Week 3: Operators & Control Structures

Operators, precedence, Boolean expressions, `if-elif-else`, match-case, truthy/falsy values, and calculators using conditionals.

Week 4: Strings & Collections

String manipulation, slicing, methods, formatting, plus lists, tuples, sets, dictionaries, comprehensions, and nested structures.

Week 5: Loops, Iterables & Comprehensions

`for`, `while`, break/continue, `range()`, `zip()`, `enumerate()`, iterables vs iterators, and mini projects using nested loops and comprehensions.

Week 6: Functions, Lambdas & Scope

Function definitions, argument types, lambdas, functional tools (`map`, `filter`, `reduce`), scope rules, and reusable modules.

Week 7: Error Handling, Debugging & File I/O

Exception handling (`try-except`), assertions, logging, `pdb`, `open()` modes, context managers, JSON handling, and a to-do list project.

Week 8: OOP Foundations

Classes, `__init__`, instance/class attributes, inheritance, encapsulation, polymorphism, dunder methods, and projects like RPG builder.

Week 9: Advanced OOP & Decorators

Decorators, `@property`, generators, context managers, abstract classes, MRO, closures, and a shopping cart mini project.

Week 10: Type Hints, Clean Code & Testing

Typing module, static analysis, clean code principles, `unittest`, `pytest`, mocking, TDD, linters (`black`, `flake8`), and automated testing for student database.

Week 11: Concurrency & Async Python

`threading`, `multiprocessing`, `asyncio`, `await`, concurrency vs parallelism, and mini projects like async downloaders and URL pingers.

Week 12: Capstone & Python 3.13 Highlights

Performance tuning, profiling, Python 3.13 features, capstone guidelines, and project options like CLI contact manager, JSON budget tracker, or file organizer.
